# Project: LLM MUD Client

**Version:** v1.0
**Created:** 2026-04-14
**Last updated:** 2026-04-14

## What This Is

A Python-based telnet MUD client designed for LLM agents to play text-based MUD games. Model-agnostic with support for multiple LLM providers (OpenAI, Anthropic, Ollama, LM Studio).

## Core Value

Enable LLMs to autonomously play MUD games through:
- Real-time telnet connectivity
- ANSI color parsing
- WebSocket API for LLM integration
- Trigger and variable systems
- Multi-provider LLM support
