  <name>Task 1: Create Conversation dataclass</name>
  <files>conversation_manager.py</files>
  <action>
Create `conversation_manager.py` with:

1. **DialogActType Enum** (5 types):
   - greeting, question, command, farewell, statement

2. **ConversationTurn dataclass**:
   - speaker: str ("agent" or "npc")
   - text: str
   - act: DialogActType
   - timestamp: float (default=time.time())

3. **ConversationStatus Enum**:
   - ACTIVE = "active"
   - PAUSED = "paused"
   - COMPLETE = "complete"

4. **Conversation dataclass**:
   - npc_name: str
   - topic: str (current discussion topic)
   - turns: List[ConversationTurn]
   - status: ConversationStatus
   - started_at: float (default=time.time())
   - last_activity: float (default=time.time())
   - to_dict() / from_dict() methods

5. **ConversationManager class** following GoalManager pattern:
   - __init__(self, conversations_file: str = "conversations.json", npc_names: List[str] = None)
   - npc_names: list of known NPC names for detection
   - conversations: Dict[str, Conversation] (keyed by npc_name)
   - _on_change_callback: Optional[Callable]
   - Callback pattern: set_on_change_callback(), _trigger_callback()

6. **CRUD Operations**:
   - start_conversation(npc_name: str, topic: str = "general") -> Conversation
   - get_conversation(npc_name: str) -> Optional[Conversation]
   - add_turn(npc_name: str, speaker: str, text: str, act: DialogActType) -> bool
   - update_topic(npc_name: str, topic: str) -> bool
   - pause_conversation(npc_name: str) -> bool
   - resume_conversation(npc_name: str) -> bool
   - complete_conversation(npc_name: str) -> bool
   - delete_conversation(npc_name: str) -> bool
   - list_conversations() -> List[Conversation] (active first, then by last_activity)
   - get_active_conversations() -> List[Conversation]

7. **NPC Detection**:
   - _detect_npc_message(text: str) -> Optional[Tuple[str, str]]
   - Pattern: Look for "NPC_NAME says:" or "NPC_NAME: " in text
   - Check against npc_names list
   - Returns (npc_name, spoken_text) or None

8. **Persistence**:
   - save_conversations() — atomic write with os.replace, prune old completed (keep last 50)
   - load_conversations() — JSON load with error handling

9. **Integration helper**:
   - get_conversation_context() -> str
   - Returns: "Currently talking to [NPC] about [topic]" for active conversations

Use the same patterns as GoalManager:
- time import for timestamps
- JSON serialization with to_dict/from_dict
- Atomic file writes with temp file + os.replace
- Callback on any state change
